# P17 - Counter + Clock Divider + 7-seg (Basys2, VHDL)

Проект P17 реализует **8-битный счётчик** с двумя режимами работы:
- **AUTO**: счётчик сам увеличивается с частотой ~1 Гц (делитель от MCLK)
- **STEP**: счётчик увеличивается по нажатию кнопки

Индикация:
- **LED[7:0]** - текущее значение счётчика (в двоичном виде)
- **7-seg** - значение в **HEX** или **DEC** (выбор переключателем)

## Частота MCLK (важно)
Basys2 может работать от **25/50/100 МГц** (зависит от JP4). Часто по умолчанию - 50 МГц.  
Если у тебя другая частота - поменяй `CLK_HZ` в коде. (см. Basys2 RM) 

## Органы управления

### Переключатели
- `SW7` - режим:
  - `0` AUTO
  - `1` STEP
- `SW6` - формат 7-seg:
  - `0` HEX (показывает `H0xx`)
  - `1` DEC (показывает `dXYZ`)
- `SW[7:0]` - данные для загрузки в счётчик (LOAD)

### Кнопки
- `BTN0` - RESET (count = 0)
- `BTN1` - STEP (+1 в режиме STEP)
- `BTN2` - HOLD (пауза/продолжение в AUTO)
- `BTN3` - LOAD (загрузить `count <= SW[7:0]`)

## Как проверить (практика)

### Пример 1: автоматический счёт
1. Поставь `SW7=0` (AUTO)
2. `SW6=0` (HEX)
3. Смотри: на 7-seg будет `H0xx`, `xx` растёт примерно раз в секунду  
   LED показывают то же число в двоичном виде.

### Пример 2: пауза
1. В AUTO нажми `BTN2` → счёт остановится
2. Нажми `BTN2` ещё раз → счёт продолжится

### Пример 3: пошаговый режим
1. Поставь `SW7=1` (STEP)
2. Нажимай `BTN1` → счёт увеличивается на 1

### Пример 4: загрузка значения
1. Выставь `SW = 00110011` (0x33 = 51)
2. Нажми `BTN3` (LOAD)
3. Увидишь на LED `00110011`, на 7-seg:
   - HEX: `H033`
   - DEC: `d051` (если `SW6=1`)
